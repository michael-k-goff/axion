# Axion

Axion is a role-playing game, done in the clasic JPRG style.

As of December 1, 2019, Axion can be played online [here](https://stupefied-poitras-059dc7.netlify.com/axion.html). It can also be downloaded and played in the browser locally; to do so, use a static file server such as Python's [SimpleHTTPServer](https://docs.python.org/2/library/simplehttpserver.html).
